# Thunder OSS Site

Marketing site for **Thunder** — Identity for humans, services, and agents.

## Live Site

[https://brionmario.github.io/thunder-oss-site/](https://brionmario.github.io/thunder-oss-site/)

## Stack

- HTML / CSS
- React 18 (via CDN, no build step)
- JSX compiled in-browser via Babel Standalone

## Development

Open `Thunder Home.html` directly in a browser — no build or install required.

## Deployment

Deployed automatically to GitHub Pages on every push to `main` via [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).
